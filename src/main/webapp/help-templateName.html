Jenkins will try to create a cloud service using the name of the template or will use an already existing one with the same name. If you want to use a different cloud service, use the advanced section and define a cloud service to use for this template.

Rules for the template name:
1.The cloud service name must start with an alphanumeric and can contain only alphanumerics and the dash (-) character.
2.Every dash (-) character must be immediately preceded and followed by an alphanumeric, i.e. no consecutive dashes are allowed.
3.The cloud service name length must be between 3 and 63 characters.
